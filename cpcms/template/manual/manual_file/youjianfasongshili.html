CanPHP开发手册--邮件发送实例

说明：
  • 类文件：CanPHP/ext/Email.class.php
  • Email.class.php仅是对phpmailer邮件发送类进行简单的封装，方便使用
  • 默认会自动加载，不需要手动include
方法：init($config)
说明：初始化配置
//邮件发送配置
$config['SMTP_HOST']='smtp.qq.com';//smtp服务器地址
$config['SMTP_PORT']=25;//smtp服务器端口
$config['SMTP_SSL']=false;//是否启用SSL安全连接，gmail需要启用sll安全连接
$config['SMTP_USERNAME']='404352772@qq.com';//smtp服务器帐号，如：你的qq邮箱
$config['SMTP_PASSWORD']='123456';//smtp服务器帐号密码，如你的qq邮箱密码
$config['SMTP_AUTH']=true;//启用SMTP验证功能，一般需要开启
$config['SMTP_CHARSET']='utf-8';//发送的邮件内容编码
$config['SMTP_FROM_TO']='404352772@qq.com';//发件人邮件地址
$config['SMTP_FROM_NAME']='CanPHP官方';//发件人姓名
$config['SMTP_DEBUG']=false;//是否显示调试信息 
//邮件发送配置结束
方法：send($mail_to,$mail_subject,$mail_body,$mail_attach=NULL)
说明：发送邮件/span>
参数：
  • $mail_to，收件人地址，类型字符串或数组，如果收件人地址是数组，表示群发
  • $mail_subject，邮件标题
  • $mail_body，邮件内容，支持html代码
  • $mail_attach，带发送的文件，类型字符串或数组，如果是数组表示同时发送多个附件
