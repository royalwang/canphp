CanPHP开发手册--目录部署

说明：
  • 目录部署不是唯一的，可以完全按照自己的想法去部署，这里只是提出一种建议的目录部署结构
index.php文件，单一入口文件
CanPHP目录，框架目录，建议改成include，更好看一些
module目录，存放模块
template目录，存放模板
public，存放图片，css及js
  • public/cimages，存放图片
  • public/css，存放css
  • public/js，存放js
data，数据缓存和日志目录
  • data/tpl_cache，存放模板缓存
  • data/db_cahce，存放数据库缓存
  • public/html_cache，存放静态页面
  • public/log，存放错误日志
admin目录，后台目录
