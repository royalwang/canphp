CanPHP开发手册--汉字转拼音类

说明：
  • 类文件：CanPHP/ext/Pinyin.class.php
  • 默认会自动加载，不需要手动include
  • 使用场景：按首字母搜索和生成拼音标题等
方法：output($str, $utf8 = true)
说明：汉字转换成拼音，并输出
参数：
  • $str，所要转化拼音的汉字
  • $utf8，所要转化拼音的汉字是否是utf8编码，默认为true，如果是gbk或gb2312编码，设置为false
使用方法：
		$py=new Pinyin();
		echo $py->output('好好学习天天向上');//输出haohaoxuexitiantianxiangshang
